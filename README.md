Code Alpha Image Gallery
A responsive image gallery built as part of my internship at Code Alpha. This project showcases layout design, image handling, and interactive UI components.
 Features
Grid-based image gallery layout
Responsive design for different screen sizes
Image preview or lightbox effect 
Hover effects and transitions
Smooth user experience
🛠️ Built With
HTML5 – Structure of the gallery
CSS3 – Grid layout and styling
JavaScript  – Interactivity
📂 Project Structure

code-alpha-image-gallery/
│── index.html
│── style.css
│── script.js 
│── images/
│── README.md
⚙️ How It Works
Images are displayed using a CSS grid or flexbox layout.
CSS handles responsiveness and styling.
JavaScript enables:
Lightbox view
Image navigation
Dynamic effects

Basic UI/UX design principles
